# EduNexus
思维编织成网

EduNexus is a Web-only AI education ecosystem platform focused on learning mastery instead of answer shortcuts.

## Current Status

This repository currently contains planning and delivery artifacts for Phase 1 (MVP):

- Product/strategy docs
- Phase 1 execution plan and task board
- OpenAPI draft for core endpoints
- Local lightweight knowledge base (vault) templates
