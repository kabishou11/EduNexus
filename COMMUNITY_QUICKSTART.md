# 学习动态社区 - 快速开始

## 快速访问

访问路径：`/community`

或点击侧边栏：**💬 学习社区**

## 5分钟上手

### 1. 查看动态流（30秒）
- 打开社区主页
- 查看推荐动态
- 切换到"关注"或"热门"标签

### 2. 发布第一条动态（1分钟）
```
1. 在页面顶部找到编辑器
2. 选择动态类型（如"学习打卡"）
3. 输入内容：
   ✅ 完成了 React Hooks 学习
   ✅ 做了 5 道算法题
4. 添加话题：#React #学习打卡
5. 点击"发布"
```

### 3. 互动动态（1分钟）
- **点赞**：点击 ❤️ 按钮
- **表情**：悬停 ❤️ 按钮，选择 👍 🎉 🔥 等
- **评论**：点击 💬 按钮，进入详情页
- **收藏**：点击 🔖 按钮

### 4. 关注用户（30秒）
- 点击任意用户头像
- 查看用户主页
- 点击"关注"按钮

### 5. 浏览话题（1分钟）
- 点击任意话题标签（如 #React）
- 查看话题下的所有动态
- 或访问"话题广场"浏览所有话题

## 示例动态

系统内置了5条示例动态：

1. **技术分享**：React Hooks 闭包陷阱

## 常见问题

**Q: 如何添加代码片段？**
A: 在编辑器中点击"添加代码"按钮，选择语言，输入代码

**Q: 如何添加话题标签？**
A: 在话题输入框输入话题名，按回车添加

**Q: 如何查看我关注的用户动态？**
A: 切换到"关注"标签页

**Q: 如何找到热门内容？**
A: 切换到"热门"标签页，或查看侧边栏热门话题

**Q: 数据保存在哪里？**
A: 当前保存在浏览器 localStorage，清除浏览器数据会丢失
