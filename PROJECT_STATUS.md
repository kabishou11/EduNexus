## 📊 项目概览

EduNexus 是一个功能完整、性能优异、设计统一的 AI 教育生态平台，集成了知识管理、学习工作区、知识星图、社交学习、协作编辑、移动端适配、PWA 支持和高级分析等功能。

## 🚀 快速开始

### 安装依赖
```bash
cd /f/1work/EduNexus
pnpm install
```

### 启动开发服务器
```bash
cd apps/web
pnpm dev
```

### 构建生产版本
```bash
cd apps/web
pnpm build
```

## 🔐 环境变量

需要配置的环境变量（`.env.local`）：

```env
# ModelScope API
MODELSCOPE_API_KEY=your_api_key

# PWA 推送通知
NEXT_PUBLIC_VAPID_PUBLIC_KEY=your_public_key
VAPID_PRIVATE_KEY=your_private_key

# 其他配置
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
