# 知识星图系统 - 增强版

> 一个功能强大的知识可视化和学习助手系统

## 🌟 核心特性

### 📊 智能可视化
- **4种节点状态**：未学习、学习中、已掌握、需复习
- **动态节点大小**：基于重要程度和连接数
- **进度环显示**：直观展示掌握程度
- **4种关系类型**：前置、相关、包含、应用

### 🤖 智能推荐
- **下一步学习推荐**：基于当前进度智能推荐
- **知识盲区识别**：自动发现学习薄弱点
- **学习路径生成**：最优路径规划
- **遗忘预警**：提醒需要复习的内容

### 📈 进度追踪
- **整体进度统计**：完成率、平均掌握度
- **状态分布分析**：各状态节点占比
- **学习时长记录**：追踪学习投入
- **复习次数统计**：巩固效果评估

### 🎨 多样布局
- **力导向布局**：自然分布，可拖拽
- **层次布局**：清晰的知识体系
- **径向布局**：突出核心知识
- **时间轴布局**：回顾学习历程

### 🎭 主题切换
- **科技风**：紫色渐变，未来感设计
- **自然风**：粉色渐变，温暖色调
- **简约风**：纯白背景，专注内容

## 🚀 快速开始

### 安装依赖
```bash
cd apps/web
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 访问页面
```
http://localhost:3000/graph
```
