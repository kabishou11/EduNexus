# 学习分析仪表板 - 快速开始

## 访问方式

### 方法 1：直接访问
在浏览器中打开：`http://localhost:3000/workspace/analytics`

### 方法 2：通过导航
1. 启动开发服务器：`cd apps/web && npm run dev`
2. 打开浏览器访问 `http://localhost:3000`
3. 在左侧边栏找到"学习分析"（带有柱状图图标）
4. 点击进入分析页面

## 主要功能

### 1. 统计卡片（顶部）
- **总学习时长**：显示近30天累计学习时间
- **文档统计**：创建和编辑的文档数量
- **练习完成**：完成的练习题数量和正确率
- **知识点**：掌握的知识点数量和平均掌握度

### 2. 图表区域（中间左侧）
切换标签查看不同图表：

#### 学习趋势
- 学习时长趋势图（折线图）
- 活动频率图（柱状图）

#### 知识掌握
- 知识点掌握度雷达图
- 多维度展示各知识点的掌握情况

#### 学习分布
- 学习领域分布饼图
- 显示不同类别的学习内容占比

#### 练习统计
- 练习正确率饼图
- 详细的正确/错误统计

### 3. 周报区域（中间右侧）
- 本周学习时长统计
- 本周成就列表
- 改进建议
- 知识点掌握排行榜

### 4. 活跃度热力图（底部）
- GitHub Contributions 风格
- 显示近12周的学习活动
- 颜色深度表示学习强度
- 悬停显示详细信息
- 统计总天数、活跃天数、最长连续天数

## 交互功能

### 周期切换
点击"本周"或"本月"按钮切换数据显示周期

### 图表切换
点击图表区域的标签切换不同的可视化视图

### 导出报告
点击右上角"导出周报"按钮下载学习报告文本文件

### 热力图交互
- 悬停在热力图的方块上查看当天的学习详情
- 颜色越深表示学习时长越长

## 主题适配

- 自动适配 EduNexus 的橙色/琥珀色主题
- 支持深色模式切换
- 响应式设计，支持移动端和桌面端

## 技术栈

- **Next.js 14**：React 框架
- **Recharts 3.8.0**：数据可视化
- **Tailwind CSS**：样式框架
- **TypeScript**：类型安全

## 故障排除

### 页面无法访问
1. 确保开发服务器正在运行
2. 检查端口 3000 是否被占用
3. 清除浏览器缓存后重试

### 图表不显示
1. 确认 recharts 已正确安装
2. 检查浏览器控制台是否有错误
3. 尝试刷新页面

### 导航链接不显示
1. 检查 AppSidebar.tsx 是否已更新
2. 重启开发服务器
3. 清除浏览器缓存
