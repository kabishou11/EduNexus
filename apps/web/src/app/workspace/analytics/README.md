# 学习分析仪表板

## 功能概述

学习分析仪表板提供全面的学习数据可视化和分析功能，帮助学生追踪学习进度，发现提升空间。

## 主要功能

### 1. 数据统计模块

- **学习时长统计**：显示日/周/月的累计学习时长
- **文档统计**：追踪创建和编辑的文档数量
- **练习完成统计**：记录完成的练习题数量和正确率
- **知识点掌握度**：评估各知识点的掌握程度

### 2. 可视化图表

使用 Recharts 库实现的交互式图表：

- **学习时长趋势图**（折线图）：展示学习时长的变化趋势
- **活动频率图**（柱状图）：显示每日学习活动次数
- **知识点掌握度雷达图**：直观展示各知识点的掌握情况
- **学习领域分布饼图**：显示学习内容的分类分布
- **练习正确率饼图**：展示练习的正确率统计

### 3. 学习活跃度热力图

类似 GitHub Contributions 的热力图：
- 显示近12周（84天）的学习活动
- 颜色深度表示学习时长
- 统计总天数、活跃天数和最长连续学习天数

### 4. 学习报告

- **周报生成**：自动生成本周学习总结
  - 学习时长统计
  - 本周成就
  - 改进建议
  - 知识点掌握排行
- **导出功能**：支持导出为文本文件

## 文件结构

```
apps/web/src/
├── app/workspace/analytics/
│   └── page.tsx                          # 分析页面主组件
├── components/analytics/
│   ├── learning-chart.tsx                # 学习图表组件
│   └── activity-heatmap.tsx              # 活动热力图组件
└── lib/analytics/
    └── stats.ts                          # 统计数据工具函数
```

## 使用方法

1. 访问 `/workspace/analytics` 路由
2. 或在侧边栏点击"学习分析"导航链接
3. 切换"本周"/"本月"标签查看不同时间段的数据
4. 点击图表标签切换不同的可视化视图
5. 点击"导出周报"按钮下载学习报告

## 设计特点

- 使用 EduNexus 的橙色/琥珀色主题
- 响应式设计，支持移动端和桌面端
- 玻璃态效果和渐变设计
- 流畅的动画和交互效果
- 支持深色模式

## 技术栈

- **Next.js 14**：React 框架
- **Recharts**：数据可视化库
- **Tailwind CSS**：样式框架
- **TypeScript**：类型安全
