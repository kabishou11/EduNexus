# 学习动态社区

类似社交媒体的学习分享平台，让学习者可以分享学习动态、交流学习心得、互相关注和互动。

## 功能特性

### 1. 动态发布
- **多种动态类型**：
  - 💬 普通动态：分享学习想法和心得
  - 📅 学习打卡：记录每日学习进度
  - 🎉 学习成果：展示学习成就和证书
  - 📝 学习笔记：分享知识库笔记
  - 📄 长文分享：深度学习心得文章

- **富文本内容**：
  - 支持文字、图片、代码片段
  - 代码高亮显示（支持多种语言）
  - 话题标签（#话题名）

### 2. 动态互动
- **点赞和反应**：6种表情反应（👍 ❤️ 🎉 🔥 💡 🚀）
- **评论系统**：支持嵌套回复和 @提及
- **收藏和转发**：保存喜欢的动态，分享给更多人
- **举报和屏蔽**：维护社区环境

### 3. 动态流
- **推荐动态**：基于兴趣和热度的智能推荐
- **关注动态**：查看关注用户的最新动态
- **热门动态**：按互动数排序的热门内容
- **话题动态**：按话题浏览相关内容
- **无限滚动**：流畅的加载体验

### 4. 用户社交
- **关注系统**：关注感兴趣的用户
- **粉丝管理**：查看关注者和关注列表
- **互相关注**：建立学习伙伴关系
- **用户推荐**：发现相似兴趣的学习者
- **用户主页**：展示个人学习动态

### 5. 话题系统
- **热门话题**：实时趋势榜
- **话题关注**：订阅感兴趣的话题
- **话题统计**：动态数量、关注人数
- **话题广场**：浏览所有话题

## 页面结构

```
/community                          # 社区主页
  ├── /post/[id]                   # 动态详情页
  ├── /topic/[name]                # 话题页面
  ├── /topics                      # 话题广场
  └── /user/[id]                   # 用户主页
```

## 核心组件

### UI 组件
- `PostCard` - 动态卡片，展示动态内容和互动
- `PostEditor` - 动态编辑器，发布各种类型的动态
- `CommentList` - 评论列表，支持嵌套回复
- `UserFollowButton` - 关注按钮，管理用户关注关系

### 数据管理
- `post-storage.ts` - 动态、评论、互动的本地存储
- `follow-system.ts` - 关注系统和用户推荐
- `feed-algorithm.ts` - 动态流推荐算法

## 推荐算法

### 推荐动态算法
```typescript
推荐分数 = 基础分数 + 互动分数 × 时间衰减

基础分数：
- 关注用户的动态：+50

互动分数：
- 点赞数 × 2
- 评论数 × 5
- 转发数 × 10
- 收藏数 × 3

时间衰减：
- 使用指数衰减函数
- 24小时半衰期
```

### 热门动态算法
```typescript
热门分数 = 点赞数 × 2 + 评论数 × 5 + 转发数 × 10
```

### 趋势动态算法
```typescript
趋势分数 = 互动总数 / (发布时间 + 2)^1.5
```

## 使用方式

1. **发布动态**：
   - 在社区主页点击动态编辑器
   - 选择动态类型
   - 输入内容，添加话题标签
   - 可选：添加代码片段
   - 点击发布

2. **互动动态**：
   - 点赞：单击点赞按钮
   - 表情反应：悬停点赞按钮，选择表情
   - 评论：点击评论按钮，输入评论内容
   - 收藏：点击收藏按钮
   - 转发：点击转发按钮

3. **关注用户**：
   - 点击用户头像进入主页
   - 点击关注按钮
   - 在"关注"标签页查看关注用户的动态

4. **浏览话题**：
   - 点击话题标签进入话题页面
   - 在话题广场浏览所有话题
   - 查看话题下的所有动态

## 技术特点

- **无限滚动**：使用 Intersection Observer API 实现流畅的无限滚动
- **本地存储**：使用 localStorage 存储数据，支持离线访问
- **实时更新**：互动操作立即反馈，无需刷新页面
- **响应式设计**：适配桌面和移动设备
- **性能优化**：虚拟滚动、懒加载、防抖节流
