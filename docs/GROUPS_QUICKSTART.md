# 学习小组系统 - 快速启动指南

## 快速开始

### 1. 启动开发服务器

```bash
cd apps/web
pnpm dev
```

### 2. 访问小组功能

打开浏览器访问：
- 小组列表：http://localhost:3000/groups
- 创建小组：http://localhost:3000/groups/create

### 3. 体验功能

#### 浏览小组
- 系统会自动生成 3 个示例小组
- 可以按分类筛选（编程、数学、语言等）
- 可以按活跃度、成员数、创建时间排序
- 支持搜索小组名称、描述和标签

#### 创建小组
1. 点击"创建小组"按钮
2. 填写小组信息：
   - 名称（2-50 字符）
   - 描述（10-500 字符）
   - 选择分类
   - 选择可见性（公开/私密）
3. 提交创建

#### 加入小组
- 点击小组卡片进入详情页
- 点击"加入小组"按钮

#### 小组功能

**讨论区**
- 发布帖子（标题 + 内容）
- 评论帖子
- 点赞帖子和评论
- 查看置顶公告

**成员管理**
- 查看所有成员
- 显示成员角色（组长/管理员/成员）
- 显示贡献度和加入时间

**任务管理**
- 创建任务（标题、描述、截止日期）
- 分配任务给成员
- 更新任务状态（待处理 → 进行中 → 已完成）
- 看板式任务展示

**资源共享**
- 添加文件、链接或笔记
- 下载共享资源
- 查看资源统计（下载量、点赞数）

**排行榜**
- 查看成员贡献度排名
- 前三名特殊展示
- 显示成员角色和贡献值

**统计分析**
- 总帖子数
- 任务完成率
- 活跃成员数
- 周活跃度

## 技术实现

### 数据存储
使用 localStorage 存储所有数据，包括：
- `edunexus_groups` - 小组列表
- `edunexus_group_members_{groupId}` - 小组成员
- `edunexus_group_posts_{groupId}` - 小组帖子
- `edunexus_group_tasks_{groupId}` - 小组任务
- `edunexus_group_resources_{groupId}` - 小组资源
- `edunexus_user_groups_{userId}` - 用户加入的小组

### 组件架构
```
GroupsPage (列表页)
  └── GroupCard (小组卡片)

CreateGroupPage (创建页)

GroupDetailPage (详情页)
  ├── GroupDiscussion (讨论区)
  ├── GroupMembers (成员列表)
  ├── GroupTasks (任务管理)
  ├── GroupResources (资源共享)
  ├── GroupLeaderboard (排行榜)
  └── Stats (统计)
```

### 类型系统
完整的 TypeScript 类型定义：
- Group - 小组信息
- GroupMember - 成员信息
- GroupPost - 帖子信息
- GroupTask - 任务信息
- GroupResource - 资源信息
- GroupStats - 统计信息

## 常见问题

**Q: 数据会丢失吗？**
A: 数据存储在浏览器的 localStorage 中，清除浏览器数据会导致数据丢失。

**Q: 如何重置示例数据？**
A: 清除浏览器的 localStorage 或删除相关键值，刷新页面会重新生成。

**Q: 支持多少成员？**
A: 默认每个小组最多 100 名成员，可在小组设置中修改。

**Q: 私密小组如何加入？**
A: 私密小组会生成唯一的邀请码，需要输入邀请码才能加入。
