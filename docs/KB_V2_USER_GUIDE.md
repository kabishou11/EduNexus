# EduNexus 知识库 V2 使用指南

## 快速开始

### 访问知识库

访问 `http://localhost:3000/kb-v2` 即可进入全新的知识库界面。

### 界面概览

知识库采用三栏布局：

```
┌─────────────┬──────────────────────┬─────────────┐
│             │                      │             │
│  左侧边栏    │    编辑区            │  右侧面板    │
│  (280px)    │    (flex-1)          │  (320px)    │
│             │                      │             │
│  - 知识库    │  - 工具栏            │  - 大纲      │
│  - 搜索      │  - 标题              │  - AI 功能   │
│  - 文档树    │  - 编辑器            │  - 属性      │
│  - 新建      │                      │  - 历史      │
│             │                      │             │
└─────────────┴──────────────────────┴─────────────┘
```

## 基础功能

### 1. 创建文档

**方法一：使用按钮**
1. 点击左侧边栏底部的"新建文档"按钮
2. 输入文档标题
3. 点击"创建"

**方法二：使用快捷键**
- 按 `Ctrl/Cmd + N` 快速创建新文档

### 2. 编辑文档

#### 标题编辑
- 点击编辑区顶部的标题区域
- 直接输入或修改标题
- 自动保存

#### 内容编辑
- 在编辑器中输入内容
- 支持 Markdown 语法
- 实时自动保存

#### 使用工具栏
工具栏提供了丰富的格式化选项：

**文本格式**
- **粗体**: 点击 B 按钮或按 `Ctrl/Cmd + B`
- *斜体*: 点击 I 按钮或按 `Ctrl/Cmd + I`
- ~~删除线~~: 点击删除线按钮
- `行内代码`: 点击代码按钮

**标题**
- H1: 一级标题
- H2: 二级标题
- H3: 三级标题

**列表**
- 无序列表
- 有序列表
- 任务列表（待办事项）

**其他**
- 引用块
- 链接
- 图片
- 表格

### 3. 搜索文档

**全局搜索**
1. 点击左侧边栏顶部的搜索框
2. 输入关键词
3. 实时显示匹配的文档

**快捷键**
- 按 `Ctrl/Cmd + K` 快速聚焦搜索框

### 4. 组织文档

#### 文档分组
左侧边栏提供了多种文档视图：

**最近访问**
- 显示最近编辑的 5 个文档
- 按更新时间排序

**所有文档**
- 显示当前知识库的所有文档
- 支持搜索过滤

**收藏夹**（计划中）
- 收藏重要文档
- 快速访问

**标签分组**（计划中）
- 按标签组织文档
- 多标签支持

### 5. 删除文档

1. 在文档列表中找到要删除的文档
2. 悬停在文档上，点击右侧的"..."按钮
3. 选择"删除"
4. 确认删除

## AI 功能

### 1. AI 思维导图

**生成思维导图**
1. 打开一个文档
2. 点击右侧面板的"AI"标签
3. 在"AI 思维导图"区域点击"生成思维导图"
4. 等待 AI 分析文档内容
5. 查看生成的思维导图

**操作思维导图**
- **缩放**: 使用鼠标滚轮或触控板
- **平移**: 拖拽画布
- **全屏**: 点击"全屏查看"按钮
- **导出**: 点击"导出"按钮（计划中）

**重新生成**
- 点击"重新生成"按钮
- AI 会重新分析文档并生成新的思维导图

### 2. AI 摘要

**生成摘要**
1. 打开一个文档
2. 点击右侧面板的"AI"标签
3. 在"AI 摘要"区域点击"生成摘要"
4. 等待 AI 生成摘要
5. 查看摘要内容

**使用摘要**
- **复制**: 点击"复制"按钮，将摘要复制到剪贴板
- **重新生成**: 点击"重新生成"按钮

### 3. AI 问答

**提问**
1. 打开一个文档
2. 点击右侧面板的"AI"标签
3. 在"AI 问答"区域的输入框中输入问题
4. 按 Enter 或点击发送按钮
5. 等待 AI 回答

**对话历史**
- 所有问答记录都会保存在对话框中
- 支持多轮对话
- AI 会基于文档内容回答问题

### 2. 知识库管理

**切换知识库**
1. 点击左侧边栏顶部的知识库名称
2. 在下拉菜单中选择其他知识库

**创建知识库**
1. 点击左侧边栏顶部的知识库名称
2. 在下拉菜单中选择"新建知识库"
3. 输入知识库名称
4. 点击"创建"

### 3. 文档属性

**查看属性**
1. 打开一个文档
2. 点击右侧面板的"属性"标签
3. 查看文档的详细信息：
   - 标签
   - 创建时间
   - 更新时间

**编辑标签**
1. 在属性面板中找到"标签"区域
2. 点击"添加标签"
3. 输入标签名称
4. 按 Enter 保存

### 4. 导出文档

**导出为 Markdown**
1. 打开一个文档
2. 点击右侧面板的"属性"标签
3. 点击"导出为 Markdown"按钮
4. 文档将下载为 .md 文件

**导出为 PDF**（计划中）
- 即将支持导出为 PDF 格式

### 5. 分享文档

**生成分享链接**（计划中）
1. 打开一个文档
2. 点击右侧面板的"属性"标签
3. 点击"分享文档"按钮
4. 设置权限（只读/可编辑）
5. 复制分享链接

## 常见问题

### 1. 文档没有自动保存？

**检查项**
- 确保浏览器支持 IndexedDB
- 检查浏览器控制台是否有错误
- 尝试刷新页面

### 2. AI 功能无法使用？

**可能原因**
- 未配置 OpenAI API Key
- API 配额已用完
- 网络连接问题

**解决方法**
- 检查环境变量配置
- 查看 API 使用情况
- 检查网络连接

### 3. 思维导图显示不正常？

**解决方法**
- 刷新页面
- 重新生成思维导图
- 检查文档内容是否过短

### 4. 快捷键不生效？

**可能原因**
- 浏览器快捷键冲突
- 输入法拦截
- 焦点不在编辑器

**解决方法**
- 关闭浏览器扩展
- 切换到英文输入法
- 点击编辑器区域

### 5. 如何备份数据？

**方法一：导出所有文档**
1. 逐个导出文档为 Markdown
2. 保存到本地文件夹

**方法二：导出数据库**（计划中）
- 一键导出所有数据
- 包含文档、标签、设置等

### 6. 如何迁移数据？

**从旧版本迁移**
- V2 自动兼容 V1 数据
- 首次访问时自动迁移
- 无需手动操作

**从其他应用迁移**
- 导出为 Markdown 格式
- 使用批量导入功能
- 保持文档结构

## 最佳实践

### 1. 文档命名

**清晰明确**
- 使用描述性标题
- 避免使用日期作为标题
- 使用关键词

**示例**
- ✅ "React Hooks 学习笔记"
- ✅ "数据结构 - 二叉树"
- ❌ "2026-03-11"
- ❌ "笔记1"

### 2. 标签使用

**分类标签**
- 学科：数学、物理、编程
- 类型：笔记、总结、问题
- 状态：进行中、已完成

**示例**
```
标签：编程, JavaScript, 学习笔记, 进行中
```

### 3. 文档结构

**使用标题层级**
```markdown
# 主题

## 第一部分
### 小节 1
### 小节 2

## 第二部分
### 小节 1
### 小节 2
```

**使用列表**
- 要点列表
- 步骤说明
- 任务清单

### 4. AI 功能使用

**思维导图**
- 适合：概念梳理、知识体系
- 文档长度：500-3000 字最佳
- 结构清晰的文档效果更好

**摘要**
- 适合：长文档、学习总结
- 文档长度：1000 字以上
- 包含明确观点的文档效果更好

**问答**
- 适合：答疑解惑、深入理解
- 提问要具体明确
- 可以多轮对话
