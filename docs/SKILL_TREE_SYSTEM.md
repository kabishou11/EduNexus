# 游戏化成长地图系统 - 功能文档

## 概述

游戏化成长地图系统将传统的学习路径重构为技能树式展示，采用游戏化机制提升学习动力和参与度。

## 核心功能

### 1. 技能树可视化

使用 React Flow 创建交互式技能树，支持：

- **节点类型**：
  - 基础技能（圆形，绿色）- 入门级技能
  - 进阶技能（六边形，蓝色）- 中级技能
  - 高级技能（星形，紫色）- 专家级技能
  - 里程碑（钻石形，金色）- 重要成就节点

- **节点状态**：
  - 🔒 未解锁（locked）- 灰色，需完成前置技能
  - ⚡ 可学习（available）- 发光动画，可以开始学习
  - ▶️ 学习中（in_progress）- 蓝色光环，显示进度环
  - ✅ 已完成（completed）- 半透明，显示完成标记

- **连接线类型**：
  - 依赖关系（橙色实线）- 必须完成的前置技能
  - 可选关系（灰色虚线）- 建议但非必需
  - 推荐关系（蓝色动画线）- AI 推荐的学习路径

### 2. 游戏化机制

#### 经验值系统
- 完成技能节点获得经验值
- 不同类型节点奖励不同：
  - 基础技能：100 EXP
  - 进阶技能：150 EXP
  - 高级技能：200 EXP
  - 里程碑：500 EXP
- 连击奖励：24小时内完成多个节点，经验值 +50%

#### 技能点系统
- 完成节点获得技能点
- 用于解锁可选技能分支
- 技能点获取规则：
  - 基础技能：1 点
  - 进阶技能：2 点
  - 高级技能：3 点
  - 里程碑：5 点

#### 成就系统
- 完成特定组合解锁成就
- 成就类型：
  - 完成度成就：完成 N 个节点
  - 速度成就：在限定时间内完成
  - 精通成就：所有练习满分
  - 探索成就：尝试多个学习路径
  - 社交成就：帮助他人、分享笔记

#### 等级系统
- 基于总经验值的等级系统
- 每个等级解锁新功能：
  - Lv 5：自定义技能树
  - Lv 10：AI 学习助手
  - Lv 15：技能树分享
  - Lv 20：高级统计分析
  - Lv 25：创建模板
  - Lv 30：专家认证

- 在线演示链接

#### 学习笔记

### 4. AI 智能规划

#### 个性化技能树生成
根据用户输入生成定制技能树：
- 学习主题
- 当前水平（beginner/intermediate/advanced）
- 学习目标
- 每周可用时间
- 学习风格（visual/practical/theoretical）

#### 学习路径推荐
AI 分析并推荐最优学习路径：
- 基于依赖关系的拓扑排序
- 考虑用户已完成的技能
- 优先推荐解锁更多后续技能的节点
- 根据时间约束调整推荐

#### 难度动态调整
根据用户表现调整难度：
- 完成速度快且分数高 → 增加难度
- 完成慢或分数低 → 降低难度，加强基础
- 实时调整预估学习时间

#### 学习建议
为每个节点生成个性化建议：
- 前置技能检查
- 时间规划建议
- 资源学习顺序
- 练习策略

#### 每周学习计划
自动生成每周学习计划：
- 根据每日可用时间分配任务
- 平衡难度和工作量
- 考虑依赖关系
- 预留复习时间

## 技术实现

### 前端技术栈
- **React Flow**：技能树可视化
- **React**：UI 组件
- **TypeScript**：类型安全
- **Tailwind CSS**：样式系统
- **Zustand**：状态管理
- **React Query**：数据获取

### 核心模块

#### 1. 类型定义 (`skill-tree-types.ts`)
定义所有数据结构：
- SkillNode：技能节点
- SkillTree：技能树
- SkillTreeProgress：学习进度
- Achievement：成就
- SkillTreeTemplate：模板

#### 2. 技能树引擎 (`skill-tree-engine.ts`)
核心逻辑处理：
- 节点解锁检查
- 状态管理
- 进度计算
- 推荐路径生成

#### 3. 解锁系统 (`unlock-system.ts`)
游戏化机制：
- 成就检查
- 技能点计算
- 连击奖励
- 解锁动画

#### 4. AI 规划器 (`ai-planner.ts`)
智能功能：
- 个性化生成
- 路径优化
- 难度预测
- 学习建议

### React 组件

#### 1. SkillNode (`skill-node.tsx`)
技能节点组件：
- 不同类型的视觉样式
- 状态指示器
- 进度环显示
- 交互动画

#### 2. SkillTreeView (`skill-tree.tsx`)
技能树视图：
- React Flow 集成
- 节点和边渲染
- 缩放和拖拽
- 小地图

#### 3. SkillDetailPanel (`skill-detail-panel.tsx`)
详情面板：
- 节点详细信息
- 学习资源列表
- 练习题展示
- 操作按钮

#### 4. PathMarket (`path-market.tsx`)
路径市场：
- 模板浏览
- 搜索筛选
- 模板预览
- 一键使用

## API 设计

### 技能树管理
- `GET /api/skill-trees` - 获取用户的技能树列表
- `GET /api/skill-trees/:id` - 获取特定技能树
- `POST /api/skill-trees` - 创建新技能树
- `PUT /api/skill-trees/:id` - 更新技能树
- `DELETE /api/skill-trees/:id` - 删除技能树

### 进度管理
- `GET /api/skill-trees/:id/progress` - 获取学习进度
- `POST /api/skill-trees/:id/nodes/:nodeId/start` - 开始学习节点
- `POST /api/skill-trees/:id/nodes/:nodeId/complete` - 完成节点
- `POST /api/skill-trees/:id/nodes/:nodeId/unlock` - 解锁节点

### 模板市场
- `GET /api/templates` - 获取模板列表
- `GET /api/templates/:id` - 获取模板详情
- `POST /api/templates/:id/use` - 使用模板
- `POST /api/templates` - 创建模板

### AI 功能
- `POST /api/ai/generate-tree` - 生成个性化技能树
- `POST /api/ai/recommend-path` - 获取学习路径推荐
- `POST /api/ai/weekly-plan` - 生成每周计划

## 使用指南

### 快速开始

1. **选择或创建技能树**
   - 从模板市场选择预设模板
   - 或创建自定义技能树

2. **开始学习**
   - 点击可用节点查看详情
   - 阅读学习资源
   - 完成练习题

3. **追踪进度**
   - 查看完成度统计
   - 获得经验值和技能点
   - 解锁新技能

4. **获得成就**
   - 完成里程碑
   - 解锁特殊徽章
   - 提升等级

### 最佳实践

1. **遵循推荐路径**：AI 推荐基于科学的学习顺序
2. **每日学习**：保持连击获得额外奖励
3. **完成练习**：实践是最好的学习方式
4. **记录笔记**：帮助巩固知识
5. **参与社区**：与他人交流学习经验

## 常见问题

**Q: 如何解锁新技能？**
A: 完成前置技能并使用技能点解锁。

**Q: 技能点如何获得？**
A: 完成任何技能节点都会获得技能点。

**Q: 可以跳过某些技能吗？**
A: 可以，但需要消耗技能点解锁可选技能。

**Q: 如何获得更多经验值？**
A: 完成节点、保持连击、获得成就都能获得经验值。

**Q: 可以自定义技能树吗？**
A: 达到 Lv 5 后可以创建自定义技能树。
