# 知识库 V2 - 全新升级 🚀

## 概述

EduNexus 知识库 V2 是一个现代化的笔记应用，参考了 Notion、飞书文档和语雀等优秀产品的设计理念。

## 主要特性

### ✨ 现代化界面
- **三栏布局**: 左侧边栏、编辑区、右侧面板
- **可折叠侧边栏**: 最大化编辑空间
- **流畅动画**: 基于 Framer Motion 的优雅过渡效果
- **响应式设计**: 完美适配各种屏幕尺寸

### 📝 强大的编辑器
- **Tiptap 编辑器**: 基于 ProseMirror 的富文本编辑器
- **Markdown 支持**: 完整的 Markdown 语法支持
- **丰富格式**: 标题、列表、表格、代码块、任务列表等
- **实时保存**: 自动保存，永不丢失数据

### 🤖 AI 增强功能
- **AI 思维导图**: 自动生成文档的可视化思维导图
- **AI 摘要**: 一键生成文档摘要
- **AI 问答**: 基于文档内容的智能问答

### ⚡ 高效操作
- **快捷键支持**: 丰富的快捷键提高效率
- **全局搜索**: 快速查找文档
- **标签管理**: 灵活的文档组织方式

## 快速开始

### 访问地址
```
http://localhost:3000/kb-v2
```

### 基本操作

1. **创建文档**: 点击左侧边栏的"新建文档"按钮或按 `Ctrl/Cmd + N`
2. **编辑文档**: 在编辑器中输入内容，支持 Markdown 语法
3. **使用 AI**: 点击右侧面板的"AI"标签，使用各种 AI 功能
4. **搜索文档**: 在左侧边栏顶部的搜索框中输入关键词

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 新建文档 |
| `Ctrl/Cmd + K` | 全局搜索 |
| `Ctrl/Cmd + B` | 切换左侧边栏 |
| `Ctrl/Cmd + \` | 切换右侧面板 |

## 技术栈

- **前端**: Next.js 14, React 18, TypeScript
- **编辑器**: Tiptap (ProseMirror)
- **UI**: Radix UI, Tailwind CSS
- **动画**: Framer Motion
- **图表**: ReactFlow
- **AI**: LangChain, OpenAI
- **存储**: IndexedDB
