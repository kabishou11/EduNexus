## 问题和建议

### 发现的问题

1. ⚠️ **相似度查询性能**: 当节点数量超过 1000 时，相似度查询耗时较长（350ms）
   - **影响**: 中等
   - **建议**: 添加缓存机制或使用 Web Worker

2. ⚠️ **批量导入**: 批量导入大量文档时，UI 可能短暂卡顿
   - **影响**: 低
   - **建议**: 使用 requestIdleCallback 分批处理

### 优化建议

1. **添加进度指示**: 批量同步时显示进度条
2. **缓存优化**: 缓存常用的图谱节点查询结果
3. **Web Worker**: 将内容提取移到 Web Worker
4. **虚拟滚动**: 大量数据展示时使用虚拟滚动
5. **懒加载**: 按需加载关联数据
