# 知识库 V2 更新日志

## [2.0.0] - 2026-03-11

### 🎉 重大更新

这是知识库的全新版本，完全重新设计和实现。

### ✨ 新增功能

#### 界面设计
- **三栏布局**: 左侧边栏（280px）、编辑区（flex-1）、右侧面板（320px）
- **可折叠侧边栏**: 支持折叠左侧边栏和右侧面板，最大化编辑空间
- **流畅动画**: 使用 Framer Motion 实现优雅的过渡动画
- **响应式设计**: 完美适配各种屏幕尺寸

#### 编辑器
- **Tiptap 富文本编辑器**: 基于 ProseMirror 的强大编辑器
- **完整的 Markdown 支持**: 支持所有常用 Markdown 语法
- **富文本格式**: 粗体、斜体、删除线、行内代码
- **标题层级**: 支持 H1-H6 六级标题
- **列表支持**: 无序列表、有序列表、任务列表
- **表格编辑**: 支持插入和编辑表格
- **代码块**: 支持代码高亮
- **图片和链接**: 支持插入图片和链接
- **引用块**: 支持引用块格式
- **实时预览**: 所见即所得的编辑体验
- **自动保存**: 防抖保存，实时同步
- **保存状态指示**: 显示保存状态和最后保存时间

#### 快捷键
- **全局快捷键**:
  - `Ctrl/Cmd + N`: 新建文档
  - `Ctrl/Cmd + K`: 全局搜索
  - `Ctrl/Cmd + B`: 切换左侧边栏
  - `Ctrl/Cmd + \`: 切换右侧面板
- **编辑器快捷键**:
  - `Ctrl/Cmd + B`: 粗体
  - `Ctrl/Cmd + I`: 斜体
  - `Ctrl/Cmd + Z`: 撤销
  - `Ctrl/Cmd + Shift + Z`: 重做

#### 左侧边栏
- **知识库切换器**: 下拉菜单选择知识库
- **全局搜索**: 实时搜索文档标题和内容
- **文档分组**:
  - 最近访问（最近 5 个文档）
  - 所有文档
- **右键菜单**: 支持删除等操作
- **新建文档**: 快速创建新文档

#### 右侧面板
- **标签页组织**: 大纲、AI、属性、历史四个标签页
- **文档大纲**: 显示文档结构（计划中）
- **AI 功能**:
  - **AI 思维导图**: 使用 ReactFlow 生成可视化思维导图
    - 支持缩放和平移
    - 支持全屏查看
    - 支持导出（计划中）
  - **AI 摘要**: 一键生成文档摘要
    - 支持复制摘要
    - 支持重新生成
  - **AI 问答**: 对话式问答界面
    - 支持多轮对话
    - 基于文档内容回答
- **文档属性**:
  - 标签管理
  - 创建时间
  - 更新时间
  - 导出和分享操作
- **版本历史**: 显示文档版本（计划中）

#### API 接口
- **思维导图生成**: `/api/kb/mindmap`
- **文档摘要**: `/api/kb/summary`
- **AI 问答**: `/api/kb/chat`

### 🔧 技术栈

- **前端框架**: Next.js 14 (App Router)
- **编辑器**: Tiptap (ProseMirror)
- **UI 组件**: Radix UI + Tailwind CSS
- **动画**: Framer Motion
- **图表**: ReactFlow
- **AI**: LangChain + OpenAI
- **存储**: IndexedDB

### 🔄 数据兼容

### 📝 迁移指南

#### 从 V1 迁移到 V2

1. **访问新版本**
   ```
   http://localhost:3000/kb-v2
   ```

2. **自动迁移**
   - 系统会自动检测并迁移 V1 数据
   - 无需手动操作

3. **测试功能**
   - 检查文档是否正常显示
   - 测试编辑和保存功能
   - 尝试 AI 功能

4. **反馈问题**
   - 如遇到问题，请在 GitHub 提交 Issue
   - 提供详细的复现步骤

#### 回退到 V1

如果需要回退到 V1：
```
http://localhost:3000/kb
```

数据不会丢失，可以随时切换。
