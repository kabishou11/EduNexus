# 数据流图

## 搜索流程

```
用户输入查询
    │
    ▼
SearchBar 组件
    │
    ├─► 保存到搜索历史 (localStorage)
    │
    ├─► 显示搜索建议 (基于文档标题和标签)
    │
    ▼
useKBSearch Hook
    │
    ├─► 防抖处理 (300ms)
    │
    ▼
SearchIndex Manager
    │
    ├─► 检查索引是否过期
    │   │
    │   ├─► 是：重建索引
    │   │   │
    │   │   ├─► 遍历所有文档
    │   │   ├─► 对标题、内容、标签分词
    │   │   └─► 构建倒排索引
    │   │
    │   └─► 否：使用缓存索引
    │
    ├─► 执行搜索
    │   │
    │   ├─► 对查询进行分词
    │   ├─► 在索引中查找匹配
    │   ├─► 计算相关性得分
    │   └─► 生成高亮片段
    │
    ▼
应用筛选条件
    │
    ├─► 标签筛选 (AND/OR)
    ├─► 日期范围筛选
    └─► 文档类型筛选
    │
    ▼
排序结果
    │
    ├─► 按相关性
    ├─► 按日期
    └─► 按标题
    │
    ▼
SearchResults 组件
    │
    ├─► 渲染搜索结果
    ├─► 高亮匹配内容
    └─► 显示元信息
```

## 索引构建流程

```
文档集合
    │
    ▼
遍历每个文档
    │
    ├─► 提取标题
    │   └─► 分词 → tokens
    │
    ├─► 提取内容
    │   └─► 分词 → tokens
    │
    ├─► 提取标签
    │   └─► 分词 → tokens
    │
    ▼
构建索引项
    │
    ├─► docId
    ├─► title
    ├─► content
    ├─► tags
    ├─► tokens (所有分词结果)
    └─► updatedAt
    │
    ▼
存储到索引 Map
    │
    └─► Map<docId, IndexItem>
```

## 相关性评分流程

```
查询词条 + 文档
    │
    ▼
初始化得分 = 0
    │
    ├─► ID 完全匹配？
    │   └─► 是：+6 分
    │
    ├─► ID 部分匹配？
    │   └─► 是：+3.5 分
    │
    ├─► 标题包含查询？
    │   └─► 是：+3 分
    │
    ├─► 标签匹配？
    │   └─► 是：+2 分/标签
    │
    ├─► 内容匹配？
    │   └─► 是：+1.5 分/次
    │
    ├─► 链接匹配？
    │   └─► 是：+1 分
    │
    └─► 新鲜度加成
        │
        ├─► 7天内：+5 分
        └─► 30天内：+2 分
    │
    ▼
最终得分
```
