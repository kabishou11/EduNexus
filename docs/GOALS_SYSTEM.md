# 目标管理系统

## 功能概述

目标管理系统帮助用户设定和追踪学习目标，养成良好的学习习惯。

## 主要功能

### 1. SMART 目标设定
- 四步向导引导用户创建 SMART 目标
- 支持目标分类（考试、技能、项目、习惯、其他）
- 支持目标类型（短期、中期、长期）
- 自动进度追踪和里程碑管理

### 2. 目标追踪
- 可视化进度条显示目标完成度
- 里程碑标记和完成状态
- 目标完成庆祝动画
- 剩余天数提醒

### 3. 习惯养成
- GitHub 风格的热力图日历
- 每日打卡功能
- 连续打卡统计（当前连续、最长连续）
- 习惯分析（完成率、月度趋势）

## 使用方法

### 创建目标
1. 访问 `/goals` 页面
2. 点击"创建新目标"按钮
3. 按照 SMART 向导填写目标信息
4. 完成后目标会显示在目标列表中

### 追踪进度
- 点击目标卡片上的"更新进度"按钮增加进度
- 点击里程碑可以标记完成状态
- 进度达到 100% 时会显示庆祝动画

### 习惯打卡
1. 切换到"习惯养成"标签
2. 点击"创建新习惯"添加习惯
3. 每天点击"打卡"按钮记录完成
4. 查看热力图和统计数据

## 数据存储

所有数据存储在浏览器的 localStorage 中：
- `edunexus_goals` - 目标数据
- `edunexus_habits` - 习惯数据

## 特色功能

- **SMART 目标向导**：清晰的四步流程，帮助用户创建高质量目标
- **GitHub 风格热力图**：直观展示习惯养成进度
- **完成庆祝动画**：目标达成时的烟花动画增强成就感
- **智能分析**：自动计算完成率、连续天数、月度趋势等
