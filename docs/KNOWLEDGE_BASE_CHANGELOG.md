# 知识宝库更新日志

## [2.0.0] - 2026-03-09

### 🎉 重大更新

这是知识宝库的重大版本更新，带来了全新的界面和功能体验。

### ✨ 新增功能

#### 界面改进
- **可折叠面板**：左右侧边栏可独立折叠，提供更灵活的工作空间
- **全屏编辑模式**：专注写作，减少干扰
- **面板控制按钮**：快速显示/隐藏侧边栏
- **响应式布局**：更好地适应不同屏幕尺寸

#### 快速记录
- **浮动按钮**：右下角闪电图标，快速访问
- **快捷键支持**：`Ctrl/Cmd + N` 快速打开
- **标签管理**：快速添加和删除标签
- **即时保存**：创建后立即保存到知识库

#### 模板系统
- **8 种预设模板**：
  - 康奈尔笔记法
  - 思维导图笔记
  - 读书笔记
  - 会议记录
  - 学习日志
  - 代码笔记
  - 项目规划
  - 空白笔记
- **模板分类**：按笔记、学习、工作、代码、自定义分类
- **模板搜索**：快速找到需要的模板
- **自定义标题**：创建时可自定义笔记标题
- **变量替换**：支持 `{{title}}`, `{{date}}`, `{{time}}` 变量

#### AI 辅助功能
- **自动关键词提取**：从文档内容中提取关键词
- **自动标签提取**：识别 `#标签` 格式的标签
- **摘要生成**：自动生成文档摘要
- **相似度计算**：基于标签和关键词计算文档相似度
- **智能推荐**：推荐相关文档
- **内容优化建议**：
  - 内容长度检查
  - 结构优化建议
  - 标签和链接建议
- **文档统计分析**：
  - 字数统计
  - 阅读时间预估
  - 段落、标题、链接、代码块统计

#### 编辑器增强
- **工具栏**：
  - 标题插入（H1, H2, H3）
  - 格式化（粗体、斜体、删除线、代码）
  - 列表（无序、有序、任务）
  - 插入（链接、图片、引用、分隔线、代码块、表格）
- **快捷键**：
  - `Ctrl/Cmd + S`：保存文档
  - `Ctrl/Cmd + N`：快速记录
  - `Ctrl/Cmd + E`：切换编辑/预览
  - `Ctrl/Cmd + B`：粗体
  - `Ctrl/Cmd + I`：斜体
  - `Esc`：退出全屏
- **实时统计**：工具栏显示字数和阅读时间
- **智能插入**：工具栏插入时自动定位光标

#### 双链增强
- **关系图可视化**：
  - 出链列表（当前文档链接到的文档）
  - 入链列表（链接到当前文档的文档）
  - 相关文档（共享标签的文档）
  - 统计信息（出链数、入链数、标签数）
- **点击跳转**：点击关系图中的文档快速跳转
- **实时更新**：关系图随文档内容实时更新

#### 文档统计面板
- **统计卡片**：
  - 字数
  - 阅读时间
  - 链接数
  - 标题数
- **网格布局**：清晰展示各项统计
- **实时更新**：编辑时实时更新统计

#### 推荐系统
- **智能推荐**：基于相似度算法推荐相关文档
- **推荐面板**：右侧面板显示推荐文档
- **快速访问**：点击推荐文档快速跳转
