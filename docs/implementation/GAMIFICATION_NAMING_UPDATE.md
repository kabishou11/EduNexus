## 设计特点

1. **保持一致性**: 所有用户可见的文本都已更新
2. **路由不变**: 保持原有的 URL 路径，不影响现有链接
3. **视觉增强**: 使用 emoji 图标增加视觉吸引力
4. **语义优化**: 新名称更贴近游戏化学习体验
5. **橙色主题**: 保持 EduNexus 的琥珀色/橙色主题风格

## 用户体验提升

- **知识宝库**: 强调知识的价值和积累感
- **知识星图**: 营造探索和发现的氛围
- **成长地图**: 突出个人成长和进步的游戏化体验

## 技术说明

- 所有更新仅涉及前端展示文本
- 后端 API 路由保持不变
- 数据结构和类型定义无需修改
- 向后兼容，不影响现有功能
