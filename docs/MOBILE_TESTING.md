## 测试工具

### 1. Chrome DevTools

#### 设备模拟
```
1. 打开 DevTools (F12)
2. 点击设备工具栏 (Ctrl+Shift+M)
3. 选择设备或自定义尺寸
4. 测试功能
```

#### 性能分析
```
1. 打开 Performance 面板
2. 点击录制
3. 执行操作
4. 停止录制
5. 分析结果
```

#### Lighthouse 测试
```
1. 打开 Lighthouse 面板
2. 选择 Mobile
3. 点击 Generate report
4. 查看评分和建议
```

### 2. 真机测试

#### iOS 设备
```bash
# 1. 确保设备和电脑在同一网络
# 2. 启动开发服务器
pnpm dev

# 3. 在 iOS 设备上访问
# http://your-ip:3000

# 4. 使用 Safari 远程调试
# Safari > 开发 > [设备名] > [页面]
```

#### Android 设备
```bash
# 1. 启用 USB 调试
# 2. 连接设备
# 3. Chrome 访问 chrome://inspect
# 4. 选择设备进行调试
```

### 3. 自动化测试

#### 单元测试
```bash
# 运行所有测试
pnpm test

# 运行特定测试
pnpm test mobile
```

#### E2E 测试
```typescript
// 使用 Playwright 进行移动端测试
import { test, devices } from '@playwright/test';

test.use({
  ...devices['iPhone 12'],
});

test('移动端导航测试', async ({ page }) => {
  await page.goto('http://localhost:3000');

  // 检查底部导航
  await expect(page.locator('nav.mobile-nav')).toBeVisible();

  // 点击导航项
  await page.click('text=知识宝库');
  await expect(page).toHaveURL(/.*kb/);
});
```

## 性能基准

### 目标指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 首屏加载 | < 3s | 首次内容绘制 |
| 交互延迟 | < 100ms | 点击到响应 |
| 滚动帧率 | 60fps | 流畅滚动 |
| 内存使用 | < 100MB | 正常使用 |
| 包大小 | < 500KB | 初始 JS |
