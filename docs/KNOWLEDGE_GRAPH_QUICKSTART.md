# 知识星图 - 快速使用指南

## 🚀 快速开始

### 第一次使用

1. **打开知识星图**
   ```
   访问：http://localhost:3000/graph
   ```

2. **了解界面布局**
   - 顶部：搜索栏、布局选择、主题切换、功能按钮
   - 中间：交互式图谱画布
   - 右侧：节点详情面板（点击节点后显示）
   - 左上：学习路径推荐（点击"学习路径"按钮）
   - 左下：进度图例和说明

3. **基本操作**
   - 🖱️ 鼠标滚轮：缩放图谱
   - 🖱️ 拖拽空白处：平移视图
   - 🖱️ 点击节点：查看详情
   - 🖱️ 悬停节点：快速预览
   - 🖱️ 拖拽节点：调整位置（力导向布局）

## 📊 理解节点状态

### 颜色含义

| 颜色 | 状态 | 说明 | 建议操作 |
|------|------|------|----------|
| 🔘 灰色 | 未学习 | 还没有开始学习 | 查看前置知识，准备学习 |
| 🟡 黄色 | 学习中 | 正在学习过程中 | 继续完成练习和笔记 |
| 🟢 绿色 | 已掌握 | 已经充分掌握 | 定期复习，保持记忆 |
| 🟠 橙色 | 需复习 | 超过2周未复习 | 尽快安排复习计划 |

### 节点大小

- **小节点**：基础知识点，连接较少
- **中等节点**：重要知识点，有一定连接
- **大节点**：核心知识点，连接众多

### 进度环

节点外围的绿色环表示掌握程度：
- 无环：0% 掌握
- 半环：50% 掌握
- 全环：100% 掌握

## 🔍 搜索和筛选

### 搜索节点

1. 在顶部搜索框输入关键词
2. 图谱自动筛选匹配的节点
3. 清空搜索框显示所有节点

**示例**：
- 搜索 "React" 显示所有包含 React 的节点
- 搜索 "Hook" 显示 Hooks 相关内容

### 类型筛选

点击顶部的类型按钮切换显示：
- **概念**：理论知识和概念
- **主题**：主题和领域
- **资源**：学习资源和工具
- **技能**：实践技能

**技巧**：可以同时选择多个类型

### 状态筛选

通过节点详情面板筛选特定状态的节点：
- 只看未学习的内容
- 关注学习中的进度
- 检查需要复习的内容

## 🎯 学习路径功能

### 查看推荐路径

1. 点击顶部"学习路径"按钮
2. 左上角显示推荐路径列表
3. 每条路径显示：
   - 路径名称
   - 难度等级（简单/中等/困难）
   - 预计时间
   - 节点数量
   - 推荐理由

### 选择路径

1. 点击感兴趣的路径卡片
2. 查看详细的学习步骤
3. 图谱上高亮显示路径
4. 蓝色连接线和粒子动画指示方向

### 开始学习

1. 在路径详情中点击"开始学习"
2. 系统引导你按步骤学习
3. 完成每个节点的笔记和练习
4. 自动更新进度

### 自定义路径

虽然系统会自动推荐，你也可以：
1. 点击起始节点
2. 查看"后续知识"
3. 规划自己的学习路径

## 📝 节点详情面板

### 打开详情

点击任意节点，右侧显示详情面板

### 面板内容

#### 1. 基本信息
- 节点名称
- 状态标签
- 创建时间

#### 2. 统计卡片
- **掌握程度**：当前掌握百分比
- **重要程度**：在知识体系中的重要性
- **笔记数量**：关联的笔记数
- **练习进度**：完成/总数

#### 3. 学习进度
- 练习完成度进度条
- 总学习时长（分钟）
- 复习次数
- 最后学习时间

#### 4. 前置知识
- 需要先掌握的知识点
- 显示每个前置节点的状态
- 点击可跳转查看

#### 5. 后续知识
- 掌握后可以学习的内容
- 按重要性排序
- 点击可跳转查看

#### 6. 相关资源
- **相关笔记**：点击查看笔记详情
- **相关练习**：点击开始练习

#### 7. 快捷操作
- **编辑节点**：修改节点信息
- **查看笔记**：跳转到笔记页面
- **开始练习**：跳转到练习页面
- **删除节点**：删除此知识点

## 🎨 布局和主题

### 切换布局

顶部布局下拉菜单提供4种布局：

#### 1. 力导向布局（默认）
- **特点**：自然分布，动态平衡
- **适用**：日常浏览，探索关系
- **操作**：可拖拽节点调整位置

#### 2. 层次布局
- **特点**：按依赖关系分层
- **适用**：查看知识体系结构
- **操作**：固定位置，不可拖拽

#### 3. 径向布局
- **特点**：以核心节点为中心
- **适用**：突出重点知识
- **操作**：固定位置，不可拖拽

#### 4. 时间轴布局
- **特点**：按创建时间排列
- **适用**：回顾学习历程
- **操作**：固定位置，不可拖拽

### 切换主题

顶部主题下拉菜单提供3种主题：

#### 1. 科技风
- 紫色渐变背景
- 未来感设计
- 适合夜间使用

#### 2. 自然风
- 粉色渐变背景
- 温暖色调
- 适合白天使用

#### 3. 简约风
- 纯白背景
- 简洁设计
- 适合专注学习

## 💡 使用技巧

### 日常学习流程

1. **早上**：打开图谱，查看今日学习计划
2. **学习时**：点击节点，查看笔记和练习
3. **学习后**：更新节点状态，记录进度
4. **晚上**：回顾今日成果，规划明日计划

### 高效学习策略

#### 1. 先易后难
- 从灰色（未学习）节点开始
- 选择小节点（基础知识）
- 逐步攻克大节点（核心知识）

#### 2. 夯实基础
- 检查前置知识是否掌握
- 绿色（已掌握）才继续下一步
- 不要跳过重要的基础

#### 3. 及时复习
- 关注橙色（需复习）节点
- 每周安排复习时间
- 使用间隔重复法

#### 4. 建立关联
- 学习时思考与其他知识的联系
- 在笔记中添加双链
- 强化知识网络

### 知识管理技巧

#### 1. 定期整理
- 每周回顾图谱
- 删除过时内容
- 更新节点信息

#### 2. 合理分类
- 使用类型标签
- 建立清晰的层次
- 避免孤立节点

#### 3. 记录详细
- 每个节点添加笔记
- 完成相关练习
- 记录学习心得

#### 4. 设定目标
- 选择学习路径
- 设定完成时间
- 追踪进度

## 🔧 常见问题解决

### 图谱显示不全

**问题**：部分节点看不到
**解决**：
1. 使用鼠标滚轮缩小视图
2. 点击顶部的"居中"按钮（Maximize2 图标）
3. 检查筛选器是否过滤了某些节点

### 节点重叠

**问题**：多个节点重叠在一起
**解决**：
1. 切换到力导向布局
2. 拖拽节点分开
3. 等待自动调整完成

### 找不到某个节点

**问题**：记得有个节点但找不到
**解决**：
1. 使用顶部搜索框
2. 输入节点名称关键词
3. 检查类型筛选器

### 学习路径不显示

**问题**：点击"学习路径"没反应
**解决**：
1. 确保至少有一些学习中的节点
2. 刷新页面重新加载
3. 检查浏览器控制台错误

### 性能问题

**问题**：图谱卡顿或加载慢
**解决**：
1. 使用筛选功能减少节点数
2. 切换到简约主题
3. 关闭学习路径显示
4. 使用更新的浏览器
