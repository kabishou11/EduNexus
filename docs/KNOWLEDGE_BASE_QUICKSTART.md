# 知识宝库快速入门

## 5 分钟上手指南

### 第一步：创建知识库

1. 打开知识宝库页面
2. 点击左上角的知识库选择器
3. 点击"创建新知识库"
4. 输入知识库名称（如"我的学习笔记"）
5. 点击"创建"

### 第二步：创建第一篇笔记

#### 方式一：使用模板（推荐）

1. 点击工具栏的"新建"按钮
2. 在模板选择器中浏览预设模板
3. 选择适合的模板（如"康奈尔笔记法"）
4. 可选：输入自定义标题
5. 点击模板卡片创建

#### 方式二：快速记录

1. 点击右下角的闪电图标
2. 或按快捷键 `Ctrl/Cmd + N`
3. 输入标题和内容
4. 添加标签（可选）
5. 点击"保存"

### 第三步：编辑笔记

1. 在左侧文档列表中选择笔记
2. 点击工具栏的"编辑"按钮
3. 使用编辑器工具栏快速插入格式
4. 编写内容
5. 按 `Ctrl/Cmd + S` 保存

### 第四步：使用双链

1. 在编辑模式下，输入 `[[`
2. 输入要链接的文档名称
3. 输入 `]]` 完成链接
4. 示例：`[[项目管理]]`
5. 在右侧面板查看链接关系

### 第五步：添加标签

1. 在内容中输入 `#` 加标签名
2. 示例：`#学习 #前端 #React`
3. 标签会自动提取并显示
4. 可以通过标签筛选文档

## 常用功能

### 搜索文档

1. 在左侧搜索框输入关键词
2. 查看搜索结果
3. 点击文档打开

### 高级筛选

1. 点击搜索框下方的"高级筛选"
2. 选择标签（支持 AND/OR 逻辑）
3. 设置日期范围
4. 查看筛选结果

### 使用 AI 助手

1. 点击右下角的 AI 助手按钮（紫色渐变）
2. 选择快捷操作或输入问题
3. 查看 AI 回复
4. 点击"插入到文档"将内容添加到笔记

### 导出笔记

1. 选择要导出的文档
2. 点击工具栏的"导出"按钮
3. 文档将下载为 Markdown 文件

### 导入笔记

1. 点击工具栏的"导入"按钮
2. 选择一个或多个 Markdown 文件
3. 文件将自动导入到当前知识库

## 快捷键速查

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + S` | 保存文档 |
| `Ctrl/Cmd + N` | 快速记录 |
| `Ctrl/Cmd + E` | 切换编辑/预览 |
| `Ctrl/Cmd + B` | 粗体 |
| `Ctrl/Cmd + I` | 斜体 |
| `Esc` | 退出全屏 |

## Markdown 语法速查

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 格式
```markdown
**粗体**
*斜体*
~~删除线~~
`行内代码`
```

### 列表
```markdown
- 无序列表项
- 无序列表项

1. 有序列表项
2. 有序列表项

- [ ] 待办事项
- [x] 已完成事项
```

### 链接和图片
```markdown
[链接文本](URL)
![图片描述](图片URL)
[[双链文档名]]
```

### 代码块
````markdown
```javascript
const hello = "world";
```
````

### 引用
```markdown
> 这是一段引用
```

### 表格
```markdown
| 列1 | 列2 | 列3 |
| --- | --- | --- |
| 内容 | 内容 | 内容 |
```

### 分隔线
```markdown
---
```

## 最佳实践

### 1. 笔记命名
- 使用清晰、描述性的标题
- 避免使用特殊字符
- 保持标题简洁（建议 50 字以内）

### 2. 标签使用
- 使用一致的标签体系
- 每篇笔记 3-5 个标签为宜
- 标签名称简短明确

### 3. 双链建立
- 链接相关主题的笔记
- 构建知识网络
- 定期检查和维护链接

### 4. 内容组织
- 使用标题划分章节
- 适当使用列表和表格
- 保持段落简洁

### 5. 定期维护
- 定期回顾和更新笔记
- 删除过时内容
- 合并重复笔记
- 导出备份

## 常见场景示例

### 场景 1：课堂笔记
1. 使用"康奈尔笔记法"模板
2. 在笔记栏记录课堂内容
3. 在线索栏标注关键词
4. 课后在总结栏写总结
5. 添加标签：`#课程名 #章节 #日期`

### 场景 2：读书笔记
1. 使用"读书笔记"模板
2. 填写书籍基本信息
3. 记录核心内容和精彩片段
4. 写下个人思考
5. 链接相关笔记：`[[作者其他作品]]`

### 场景 3：项目管理
1. 使用"项目规划"模板
2. 明确项目目标和时间线
3. 分解任务和分配责任
4. 使用任务列表跟踪进度
5. 定期更新项目状态

### 场景 4：学习日志
1. 使用"学习日志"模板
2. 每天记录学习内容
3. 总结重要收获
4. 记录遇到的问题
5. 链接相关学习资料

### 场景 5：技术笔记
1. 使用"代码笔记"模板
2. 描述遇到的问题
3. 记录解决方案和代码
4. 标注关键要点
5. 添加相关资源链接
