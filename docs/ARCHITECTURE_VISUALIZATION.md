```

  │
```

## 错误处理流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                        错误处理流程                              │
└─────────────────────────────────────────────────────────────────┘

任务执行
    │
    ├─→ 成功
    │   ├─→ 从队列移除
    │   └─→ 记录成功日志
    │
    └─→ 失败
        │
        ├─→ 记录错误日志
        │
        ├─→ retryCount < maxRetries?
        │   │
        │   ├─→ 是
        │   │   ├─→ retryCount++
        │   │   ├─→ 延迟重试 (2s * retryCount)
        │   │   └─→ 重新入队
        │   │
        │   └─→ 否
        │       ├─→ 标记为永久失败
        │       ├─→ 记录失败日志
        │       └─→ 通知用户
        │
        └─→ 错误类型处理
            │
            ├─→ 网络错误
            │   └─→ 等待在线后重试
            │
            ├─→ 数据库错误
            │   └─→ 记录日志，跳过
            │
            ├─→ 数据格式错误
            │   └─→ 记录日志，跳过
            │
            └─→ 超时错误
                └─→ 增加超时时间，重试
```
