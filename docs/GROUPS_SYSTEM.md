# 学习小组系统

## 功能概述

学习小组系统是 EduNexus 的核心功能之一，支持用户创建和加入学习小组，进行协作学习。

## 主要功能

### 1. 小组管理
- **创建小组**: 设置名称、描述、分类、可见性（公开/私密）
- **加入小组**: 搜索公开小组或使用邀请码加入私密小组
- **成员管理**: 支持组长、管理员、成员三种角色
- **小组设置**: 配置发帖权限、邀请权限、审批要求等

### 2. 小组协作
- **讨论区**: 发布帖子、评论、点赞，支持置顶和公告
- **任务管理**: 创建任务、分配成员、跟踪进度（待处理/进行中/已完成）
- **资源共享**: 上传文件、分享链接、发布笔记
- **排行榜**: 根据贡献度展示成员排名

### 3. 统计分析
- 总帖子数
- 任务完成率
- 活跃成员数
- 周活跃度

## 使用方法

### 访问小组列表
访问 `/groups` 查看所有公开小组，可以按分类筛选和排序。

### 创建小组
1. 点击"创建小组"按钮
2. 填写小组信息（名称、描述、分类）
3. 选择可见性（公开/私密）
4. 提交创建

### 加入小组
- 公开小组：直接点击"加入小组"按钮
- 私密小组：需要输入邀请码

### 小组功能
进入小组后，可以使用以下功能：
- **讨论**: 发布帖子、参与讨论
- **成员**: 查看成员列表和角色
- **任务**: 创建和管理小组任务
- **资源**: 共享学习资料
- **排行榜**: 查看成员贡献度排名
- **统计**: 查看小组活动数据

## 数据存储

系统使用 localStorage 存储数据，包括：
- 小组信息
- 成员数据
- 帖子和评论
- 任务列表
- 共享资源
- 打卡记录
