# 知识宝库多入口快速创建系统 - 功能演示

## 演示概览

本文档提供知识宝库多入口快速创建系统的功能演示说明，帮助用户快速了解和使用新功能。

## 演示场景

### 场景 1: 使用全局快捷键快速创建

**演示步骤：**

1. 在知识库页面的任意位置
2. 按下 `Ctrl+Shift+N`（Mac: `⌘⇧N`）
3. 快速创建对话框自动弹出
4. 输入标题："学习笔记 - React Hooks"
5. 输入内容：
   ```
   # React Hooks 学习笔记

   ## useState
   - 用于在函数组件中添加状态
   - 返回状态值和更新函数

   ## useEffect
   - 用于处理副作用
   - 可以模拟生命周期方法

   #React #前端 #学习笔记
   ```
6. 系统自动提取标签：React、前端、学习笔记
7. 点击"创建"按钮
8. 笔记创建成功，自动显示在文档列表中

**预期结果：**
- 快捷键响应迅速（< 100ms）
- 对话框平滑弹出
- 标签自动提取正确
- 笔记成功创建并显示

---

### 场景 2: 使用浮动按钮创建

**演示步骤：**

1. 注意页面右下角的浮动创建按钮（橙色圆形按钮）
2. 点击浮动按钮
3. 菜单展开，显示三个选项：
   - 空白笔记（蓝色）
   - 使用模板（绿色）
   - 快速记录（紫色）
4. 选择"使用模板"
5. 快速创建对话框打开，显示模板选择界面
6. 选择"会议记录"模板
7. 填写标题："2026-03-09 团队周会"
8. 内容自动填充模板结构
9. 添加标签："会议"、"团队"
10. 点击"创建"按钮

**预期结果：**
- 浮动按钮位置合理，不遮挡内容
- 菜单展开动画流畅
- 模板内容正确填充
- 笔记创建成功

### 场景 3: 右键菜单保存选中文字

**演示步骤：**

1. 打开一个现有文档
2. 选中一段重要文字：

   可以大大提高开发效率和代码可维护性。

4. 右键菜单弹出，显示"保存到知识宝库"选项
5. 点击"保存到知识宝库"
6. 快速创建对话框打开
7. 内容区域已自动填充选中的文字
8. 系统自动生成标题："React 的核心思想是组件化和声明式编程"
9. 添加标签："React"、"核心概念"
10. 点击"创建"按钮

**预期结果：**
- 右键菜单正确显示
- 选中文字正确传递
- 标题自动生成合理
- 笔记创建成功

## 常见演示问题

### Q: 演示时快捷键不响应？

**A**:
1. 检查是否有其他应用占用快捷键
2. 确认浏览器窗口处于激活状态
3. 刷新页面重试

### Q: 对话框动画卡顿？

**A**:
1. 检查浏览器性能
2. 关闭其他占用资源的标签页
3. 使用 Chrome 或 Edge 浏览器

### Q: 标签提取不准确？

**A**:
1. 确保内容中使用 # 标签格式
2. 标签应该是单个词，不包含空格
3. 可以手动添加或修改标签
